		closeItem
	
 
	
	

Close an existing open item.


To close an item, you have to specify a valid reason. Below a list of  accepted reasons: 
  • TestItem
  • LostOrBrokenItem
  • NotAvailableItem
  • IncorrectItem

	
    
				
  • return: first element is the status (bool) and second one is additional data
    • 
				
  • access: public
    • 
			

	
	

		array
		
			closeItem
		
					(string $token, int $idItem, string $reason)
			

	
			
    
					
  • 
				string
				$token: secret token			
    • 
					
  • 
				int
				$idItem: id of the item			
    • 
					
  • 
				string
				$reason: valid reason which you want to close this item
